# GTAVC-CruiseControl
Cruise control for GTA Vice City

Special thanks to Dutchy3010 for cruise control mod. I had modified for GTA Vice City.
Special thanks to ThirteenAG for walk mod.

----------- Installation
First you have to install CLEO. You can download it here. Copy (.cs) file in the package to CLEO folder.

----------- How To Use
In Car:
 F5 : Activate/Deactivate Mod
 +  : Increase max speed
 -  : Decrease max speed
TAB : Activate/Deactivate Keep Speed. Car will keep speed without press gas button. Press again or press break button to deactivate.
 1  : Turn-off machine

On Foot:
Press TAB to activate/deactivate walk mod.
